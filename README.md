# HEHE TIERS Bot

Sistema completo de filas, testes, tiers e blacklist para o servidor HEHE TIERS.

---

# Como usar

1. Clone o repositório
2. Rode "npm install"
3. Preencha os IDs no config.yaml
4. O arquivo principal do bot é o "index.js"
5. Rode "npm start" ou "node index.js"

---

# Funcionalidades

- Sistema de filas com pings automáticos
- Cooldown de 7 dias por fila
- Blacklist com expiração automática
- Registro de rounds (normal e High Test)
- Painel de status do bot
- Auto-fechamento de filas inativas

---

Observação

- Nunca compartilhe o "config.yaml" ou o token do bot
